# configs

Personal configuration files for tools I use daily. Run `make` to sync all configs into this repo from their source locations.

## What's inside

- **ghostty** — terminal emulator config
- **helix** — editor config
- **tmux** — terminal multiplexer config + plugin list
- **lazygit** — git UI config

## Usage

```sh
make          # sync all configs
make ghostty  # sync only ghostty
make helix    # sync only helix
make tmux     # sync only tmux
make lazygit  # sync only lazygit
```
