# tmux

## Plugin Manager

Install [TPM](https://github.com/tmux-plugins/tpm):

```sh
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

Then press `prefix + I` inside tmux to install plugins.

## Plugins

- [tpm](https://github.com/tmux-plugins/tpm) — plugin manager
- [tmux-sensible](https://github.com/tmux-plugins/tmux-sensible) — sensible defaults
- [tmux-resurrect](https://github.com/tmux-plugins/tmux-resurrect) — persist sessions across restarts
- [tmux-continuum](https://github.com/tmux-plugins/tmux-continuum) — auto-save/restore sessions
- [tmux-yank](https://github.com/tmux-plugins/tmux-yank) — copy to system clipboard
- [tmux-sidebar](https://github.com/tmux-plugins/tmux-sidebar) — file tree sidebar
